# 🎬 Kinogo - Киноагрегатор

Сайт-агрегатор фильмов, подобный Кинопоиску, созданный с использованием FastAPI (Backend) и React (Frontend).

## 📁 Структура проекта

```
Kinogo/
├── backend/          # FastAPI сервер
├── frontend/         # React приложение
└── README.md
```

## 🛠 Технологический стек

### Backend
- **FastAPI** - современный фреймворк для создания API
- **Python 3.8+**
- **TMDb API** - источник данных о фильмах
- **CORS** - для взаимодействия с frontend

### Frontend
- **React 18** - фреймворк для создания пользовательского интерфейса
- **Axios** - для HTTP запросов
- **React Router** - для навигации
- **Tailwind CSS** - для стилизации

## ⚡ Функционал

### MVP (Минимально жизнеспособный продукт)
- 📱 Каталог фильмов с постерами и рейтингом
- 🔍 Поиск по названию
- 📄 Страница фильма с подробной информацией
- ⭐ Просмотр рейтингов и отзывов

### Будущий функционал
- 👤 Регистрация и авторизация пользователей
- 📝 Возможность ставить оценки и писать отзывы
- 📋 Персональные списки пользователей
- 🔽 Фильтрация и сортировка
- 🎯 Рекомендации

## 🚀 Быстрый старт

### 1. Настройка Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

**Получите API ключ TMDb:**

1. Зарегистрируйтесь на [themoviedb.org](https://www.themoviedb.org/)
2. Перейдите в настройки API и получите ключ
3. Добавьте ключ в файл `.env`:
   ```env
   TMDB_API_KEY=your_actual_api_key_here
   TMDB_BASE_URL=https://api.themoviedb.org/3
   ```

**Запустите сервер:**

```bash
uvicorn main:app --reload
```

Backend будет доступен на http://localhost:8000

### 2. Настройка Frontend

```bash
cd frontend
npm install
npm start
```

Frontend будет доступен на http://localhost:3000

### 3. Проверка работы

1. Откройте http://localhost:3000 в браузере
2. Вы должны увидеть главную страницу с популярными фильмами
3. Попробуйте поиск фильмов в шапке сайта
4. Кликните на любой фильм для просмотра детальной информации

- `GET /api/movies/popular` - популярные фильмы

- `GET /api/movies/{movie_id}` - детальная информация о фильме
