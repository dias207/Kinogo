# Frontend - Киноагрегатор

React приложение для отображения каталога фильмов с современным интерфейсом.

## Установка и запуск

1. **Установите Node.js 16+** (если еще не установлен)

2. **Установите зависимости:**
```bash
npm install
```

3. **Запустите приложение:**
```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

## Функционал

### Основные возможности:
- 🎬 **Каталог популярных фильмов** - просмотр популярных фильмов с пагинацией
- 🔍 **Поиск фильмов** - поиск по названию с результатами в реальном времени
- 📱 **Адаптивный дизайн** - корректное отображение на всех устройствах
- ⭐ **Рейтинги** - цветовая индикация рейтинга фильмов
- 🎨 **Современный UI** - темная тема с Tailwind CSS

### Страницы:
- **Главная** (`/`) - популярные фильмы
- **Поиск** (`/search`) - результаты поиска
- **Детали фильма** (`/movie/:id`) - полная информация о фильме

## Технологии

- **React 18** - библиотека для создания пользовательских интерфейсов
- **React Router** - маршрутизация в приложении
- **Axios** - HTTP клиент для запросов к API
- **Tailwind CSS** - утилитарный CSS фреймворк
- **PostCSS** - инструмент для обработки CSS

## API интеграция

Frontend взаимодействует с бэкендом по адресу `http://localhost:8000` через следующие эндпоинты:
- `/api/movies/popular` - получение популярных фильмов
- `/api/movies/search` - поиск фильмов
- `/api/movies/:id` - детальная информация

## Разработка

### Добавление новых компонентов:
1. Создайте компонент в `src/components/`
2. Используйте Tailwind CSS для стилизации
3. Следуйте существующей структуре кода

### API запросы:
- Все API запросы обернуты в `src/services/api.js`
- Используйте `try/catch` для обработки ошибок
- Добавляйте загрузочные состояния для лучшего UX
