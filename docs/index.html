Как запустить проект

Backend Setup

$ cd backend
$ python -m venv venv
$ venv\Scripts\activate
$ pip install -r requirements.txt
$ uvicorn main:app --reload

Backend будет доступен на http://localhost:8000

Frontend Setup

$ cd frontend
$ npm install
$ npm start

Frontend будет доступен на http://localhost:3000
